What are the Ground X Data Services?

Packaged datasets, APIs, and data governance rules that facilitate secure access to data from the Ground X Platform. There are two data services:

  • The Gepspatial Assets Data Service provides API access to photos and videos from cameras installed across NEOM regions.
  • The Media Assets Data Service provides API access to data from your Workspace in the Ground X Digital Asset Management (DAM) system. Note that this service is only available to teams using the Ground X DAM

Where are the cameras feeding the Geospatial Assets Data Service?

The Ground X Documentation team captures the development progress of NEOM's regions in high quality photos and videos using the following devices:

  • Timelapse cameras
  • Drone-mounted cameras
  • Helicopter-mounted cameras
  • Live cameras
  • 360-degree cameras
  • Underwater cameras

These devices record media in a short frequency of 5-30mins (drone and helicopter flights are less frequent) and transmit in real-time into Ground X. The interactive map below shows the locations of these devices.

Although most devices belong to the Ground X Unit, some cameras are installed and managed by other NEOM teams using Ground X to capture and distribute their camera feeds.

How can I access the data services?

To access the data services, simply send an email to the Ground X team at groundxsupport@neom.com with a description of your use case. We will be in touch to facilitate your data access request and authorization from data owners.

Where can I get API authorization tokens to test the APIs?

To request on authorization token for the Ground X Data Service APIs, send an email to groundxsupport@neom.com and the ream will provide the token within 24 hours.

How can I publish my own data to the Ground X Data Services?

Via the Ground X DAM UI: Simply upload your data into your Workspace in the Ground X DAM.

Via the Ground X Ingestion APIs: To automate your data transfer into Ground X, integrate your application or device with the Ground X Ingestion APIs.

How can I request the Ground X Ingestion API?

To request the Ingestion API, send an email to groundxsupport@neom.com with a description of your use case.
